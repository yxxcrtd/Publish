安装准备和系统需求

为了安装发布系统，您需要检查一下您的机器环境：

1. Windows 操作系统正常可用，我们建议您使用 Server 版本的 Windows 操作系统，如 Windows 2000 Server Edition.
请确定 Windows 操作系统已经升级到最新的 Service Pack.
Internet Explorer 在管理的时候会使用，请确定 IE 升级到 5.0 以上，并且安装了安全补丁。
（在 IE 的工具菜单中有 Windows Update 菜单项，我们建议您使用 Windows Update 更新最新补丁）

2. 确保机器上面没有病毒，没有流氓软件。
安装之前，您可以使用 Norton、瑞星、KV3000 等杀毒软件扫描一下机器，保证机器内没有病毒。
也最好使用反流氓软件扫描软件如 安全卫士360，扫描一下是否有流氓软件。
病毒和流氓软件可能导致您的系统部分组件运行不正常，或者性能下降，导致发布系统安装之后达不到预期效果，甚至无法安装。

3. 检查您是否有 SQL Server 数据库已经安装，检查您是否有 Tomcat 已经安装了，检查是否有 JRE 或者 Java SDK 安装。
如果没有安装 SQL Server 数据库服务，请先安装 SQL Server。
如果没有安装 Tomcat，则我们的安装程序会自动安装 Tomcat，您也可以手工安装 Tomcat 及其支持 JRE 系统。
如果已经安装了 Tomcat，请确保其能够正常运行，关于 Tomcat 的帮助，请查阅这里(TODO)。
如果这些都有，请记录下这些系统的目录位置。

4. 有足够的硬盘和内存。
发布系统初始状态大约是 7M 左右的数据库，16M 左右的网站系统文件。随着使用，这些都会不断增大，请确保您安装的数据库目录所在硬盘和网站所在硬盘有足够的硬盘。
发布系统在运行时要使用较多的内存，内存越多性能越好，我们推荐您至少有 256M 内存。当然越多越好。

6. 开始安装，请进入到常见问题页面，遇到安装问题时可以快速查找一下是否有这些问题的解答。
